================================
PANDUAN CEPAT
================================

📍 LANGKAH 1: SETUP (Jika belum)
   Windows: setup.bat
   Linux/Mac: bash setup.sh
   atau Manual: php artisan serve

📍 LANGKAH 2: INSTALL AUTHENTICATION
   composer require laravel/breeze --dev
   php artisan breeze:install blade
   npm install && npm run dev
   php artisan migrate

📍 LANGKAH 3: JALANKAN SERVER
   php artisan serve
   Buka: http://localhost:8000

📍 LANGKAH 4: LOGIN & MULAI
   Register akun baru
   Login
   Klik Admin → Tambahkan projects, skills, experiences

⚡ COMMAND CHEAT SHEET
================================

Start Server:
$ php artisan serve

Fresh Database:
$ php artisan migrate:fresh
$ php artisan db:seed

Storage Link (untuk gambar):
$ php artisan storage:link

Debug/Testing:
$ php artisan tinker

Cache Clear:
$ php artisan cache:clear

================================
❌ TROUBLESHOOTING
================================

Port 8000 sudah digunakan?
$ php artisan serve --port=8001

Database error?
$ php artisan migrate:fresh

Gambar tidak tampil?
$ php artisan storage:link

File tidak ditemukan?
Cek nama file (case-sensitive) di folder resources/views/
